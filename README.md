# TestNX

Seeing if AI can make Switch homebrew.

## Included example

This repository now includes a minimal Nintendo Switch homebrew example app:

- [`examples/switch/hello-world`](examples/switch/hello-world) — a libnx console
  app that prints `Hello World!` and exits when `+` is pressed.

The example is based on the public-domain `graphics/printing/hello-world`
example from [`switchbrew/switch-examples`](https://github.com/switchbrew/switch-examples).
